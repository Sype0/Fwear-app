# Fwear

**Fwear** is a minimalist, unofficial F-Droid client designed specifically for **Wear OS**.

## Installation

### Using ADB
1.  Enable **Developer Options** and **ADB Debugging** on your watch.
2.  Connect your watch to your PC via Wi-Fi.
3.  Install the APK:
    ```bash
    adb install fwear.apk
    ```
